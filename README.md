# Sales Performance & Revenue Analysis

This project focuses on analyzing sales data to understand revenue trends,
product performance, customer segments, and regional sales behavior using
SQL and Python.

---

## Tools Used

- SQL (Data analysis and KPI calculation)
- Python (Pandas, Matplotlib)
- Jupyter Notebook

---

## Project Work

- Loaded and cleaned the dataset
- Fixed date format issues (DD/MM/YYYY)
- Created new features such as:
  - Order Year
  - Order Month
  - Shipping Days
- Performed analysis using SQL queries
- Created visualizations in Python for:
  - Category-wise sales
  - Region-wise sales
  - Customer segment performance
  - Monthly and yearly sales trends
  - Top products and customers
  - Shipping time analysis
